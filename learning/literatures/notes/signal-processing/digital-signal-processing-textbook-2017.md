## Overview
A DSP textbook (Elsevier, DOI 10.1016/B978-0-12-804547-3) with 12 chapters plus appendices, covering the full spectrum of digital signal processing: signals, systems, filtering (FIR/IIR), spectral analysis, multirate processing, and MATLAB throughout. Chapter 1 introduces deterministic vs. random signals, sampling, quantisation, and basic MATLAB DSP operations.

## Unique contribution
Provides a modern (2017) DSP textbook with strong MATLAB integration, covering both fundamentals and advanced topics (adaptive filtering, multirate, DSP system design). Chapter-by-chapter organisation allows targeted reference reading.

## Key concepts
- Deterministic vs. random signals
- Sampling, Nyquist, aliasing
- Quantisation
- FIR and IIR filter design
- DFT, FFT
- Spectral analysis
- Multirate signal processing
- MATLAB DSP System Toolbox
- Convolution, correlation

## Methods / results / takeaways
- Chapter 1: signal types (deterministic/random, continuous/discrete/digital), typical DSP system (ADC → DSP → DAC), MATLAB basics (stem, ones, zeros, filter, conv, fir1, butter, cheby1, cheby2, ellip, residuez, zplane).
- Follows standard DSP curriculum: FIR filter design (windowing, Parks-McClellan), IIR filter design (bilinear transform), Z-transform, DFT/FFT, STFT.
- MATLAB scripts provided for each concept; examples use audio, vibration, and communication signals.
- Appendices cover mathematical background (A–G).
