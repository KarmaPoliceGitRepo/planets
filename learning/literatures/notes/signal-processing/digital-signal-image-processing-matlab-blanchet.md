## Overview
A textbook (translated from French, Hermès Science 2001 / ISTE 2006) covering digital signal and image processing with MATLAB implementations. Topics span 1D and 2D signal processing: sampling, Fourier transforms, filtering, spectral estimation, stochastic processes, image processing (filtering, restoration, segmentation, compression), and MATLAB exercises throughout.

## Unique contribution
Integrates signal processing theory with MATLAB code throughout; covers both 1D signal processing (DSP fundamentals) and 2D image processing in a single volume, with a French academic perspective emphasising mathematical rigour alongside practical MATLAB implementation.

## Key concepts
- Sampling, Nyquist theorem
- Discrete Fourier Transform (DFT), FFT
- FIR and IIR filter design
- Spectral estimation (periodogram, Welch, AR/MA methods)
- Stochastic processes, power spectral density
- 2D Fourier transform, image filtering
- Image restoration, deconvolution
- Image segmentation
- Image compression (DCT, wavelets)
- MATLAB implementation

## Methods / results / takeaways
- Covers standard DSP pipeline: sampling → quantisation → DFT/FFT → filter design → spectral analysis.
- Filter design methods: windowed sinc (FIR), Parks-McClellan (equiripple FIR), Butterworth/Chebyshev/elliptic (IIR).
- Spectral estimation: periodogram, averaged periodogram (Welch), parametric methods (AR, MA, ARMA).
- Image processing section covers 2D convolution, frequency domain filtering, Wiener filter restoration, segmentation (thresholding, region growing, watershed), DCT-based compression, wavelet compression.
- Each section accompanied by MATLAB scripts for implementation and visualisation.
