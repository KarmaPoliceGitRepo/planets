## Overview
Brillouin's classic monograph analysing wave propagation in dispersive media, distinguishing between phase velocity, group velocity, signal velocity, and energy velocity. Contains the original analysis showing that group velocity can exceed c without violating causality, and the resolution via the signal velocity concept. A foundational reference for dispersive wave physics.
