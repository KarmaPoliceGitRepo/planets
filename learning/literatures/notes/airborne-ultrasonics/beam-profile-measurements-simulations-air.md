## Overview
Presents measured and simulated 3D beam profiles of ultrasonic transducers operating in air. Validates finite element and angular spectrum simulation methods against experimental beam maps, providing design guidance for air-coupled transducer aperture and frequency selection.

## Unique contribution
Systematic 3D beam profile characterisation in air using scanning microphone measurements, combined with FEM and angular spectrum method simulations. Provides validated simulation tools for predicting air-coupled beam patterns — directly useful for designing arrays, predicting lateral resolution, and optimising transducer geometry.

## Key concepts
- Beam profile / radiation pattern
- Air-coupled transducer characterisation
- Finite element simulation
- Angular spectrum method

- Lateral resolution
- Transducer aperture design

## Methods / results / takeaways
- Experimental: scanning microphone maps beam pressure in 3D planes at various distances.
- Simulations: finite element method (FEM) and angular spectrum method (ASM) both compared.
- Good agreement between FEM/ASM and measurements in both near- and far-field regions.
- Near-field structure (pressure oscillations) accurately predicted; side-lobe positions matched.
- Key finding: both simulation approaches are reliable for predicting air-coupled beam profiles, enabling design optimisation without extensive prototyping.
- Useful for predicting lateral resolution in air-coupled scanning systems.
