# Ultrasonic Presence Sensors for Wide-Range Industrial Use

## Overview
Describes RU-type modular air-coupled ultrasonic presence sensors using glass-sphere/epoxy composite matching layers (Zm = 1.3×10⁶ kg/m·s). A modular system operating at 400/200/80 kHz serves different range requirements, commercialised as Siemens Sonar Bero product line.

## Unique contribution
Introduces glass-sphere-loaded epoxy as a practical bulk matching layer with controlled intermediate acoustic impedance, enabling wideband efficient coupling from PZT ceramics to air. Describes the modular commercial transducer family spanning 80–400 kHz for ranges from centimetres to several metres.

## Key concepts
- Air-coupled presence sensing
- Glass-sphere/epoxy composite matching layer
- RU-type transducers
- Modular ultrasonic sensor design
- Sonar Bero (Siemens commercial product)
- 80 / 200 / 400 kHz operating frequencies
- Impedance matching

## Methods / results / takeaways
- Matching material: glass microspheres in epoxy, Zm ≈ 1.3×10⁶ kg/m·s (between air ~430 Ra and PZT ~30 MRa).
- Three frequency variants: 400 kHz (short range, narrow beam), 200 kHz (medium), 80 kHz (long range, wide coverage).
- Modular housing allows mix-and-match for industrial automation.
- Sonar Bero commercial product based on this design achieved widespread industrial deployment.
- Ring-down time managed via damping backing and matching layer; blind zones quoted per variant.
- Key insight: a single matching layer of intermediate impedance achieves 20–30 dB improvement in insertion loss vs bare PZT in air.
