## Overview
A comparative spreadsheet collating transducer specifications and ranging accuracy from multiple published sources. Columns include acoustic impedance, insertion loss (IL), bandwidth, ring-down time, and ranging accuracy, enabling direct cross-source comparison.

## Unique contribution
Provides a curated, structured cross-reference of air-coupled transducer performance across the literature — useful as a quick-reference table that condenses many papers into a single comparable format.

## Key concepts
- Transducer comparison
- Acoustic impedance
- Insertion loss (IL)
- Bandwidth
- Ring-down time
- Ranging accuracy
- Air-coupled transducer specifications

## Methods / results / takeaways
- Columns: transducer type, frequency, acoustic impedance (Rayleigh), insertion loss (dB), −6 dB bandwidth, ring-down cycles, ranging accuracy.
- Sources span piezoelectric (PZT, composites) and capacitive (electrostatic, CMUT) types.
- Typical insertion losses range from ~20 dB (well-matched) to >60 dB (bare PZT in air).
- Ring-down time is a key metric limiting minimum measurable distance; composites and capacitive types show shorter ring-down.
- Accuracy column spans from sub-mm (high-frequency, short-range) to several mm (low-frequency, long-range).
- Useful for selecting transducer type for a given application requirement.
