# Characterization of the Interface Roughness of Coatings Based on Ultrasonic Reflection Coefficient Amplitude Spectrum

- **Source:** Ma 2017 coating interface roughness URCAS
- **Drive link:** https://drive.google.com/file/d/1Jp6JK73iXxl1FYSIfKRyQ7dmqnGohWil/view
- **Type:** paper
- **Author/Year:** Ma, Zhang, Gao, Lin & Krishnaswamy, 2017 (AIP Conference Proceedings, 1806, 090021)
- **Coverage:** full

## Overview
Presents a technique for non-destructively characterising the interface roughness of sprayed coatings using the ultrasonic reflection coefficient amplitude spectrum (URCAS). Derives the URCAS including roughness via phase-screen approximation theory, then uses two-parameter inversion (Rq and coating thickness d) with a cross-correlation algorithm. Valid for Rq = 1.5%λ to 11%λ; confirmed with FDTD simulations and water-immersion experiments on three Ni specimens (Rq = 8.5, 14.2, 28.6 μm).

## Unique contribution
Addresses the previously unmet need to characterise subsurface interface roughness (not just surface roughness) of coatings, where time-domain separation of surface and interface echoes is impossible. Reformulates URCAS for rough interfaces using phase-screen approximation, enabling simultaneous extraction of interface Rq and coating thickness d from broadband frequency-domain data.

## Key concepts
- Ultrasonic reflection coefficient amplitude spectrum (URCAS)
- Phase-screen approximation theory
- Two-parameter inversion (Rq + coating thickness d)
- Cross-correlation algorithm for parameter estimation
- Coating interface roughness NDE
- FDTD numerical simulation
- Water-immersion pulse-echo
- Valid range: Rq = 1.5%λ to 11%λ

## Methods / results / takeaways
- Technique: water-immersion, flat transducer (17 MHz dominant, −6 dB range 9.7–26 MHz); coating: sprayed Ni on substrate; three media structure (water/Ni/water).
- Model: phase-screen roughness modification of reflection coefficient R₂₃ = R₂₃⁰ × exp(−2Rq²k²z); URCAS derived as function of f, Rq, d.
- Inversion: cross-correlation coefficient η(Rq, d) maximised to find Rq and d simultaneously.
- Valid range: errors < 10% when 1.5%λ < Rq < 11%λ; errors rise outside this range.
- Effect of beam coverage: errors increase for smaller transducer diameter at high Rq; recommend D ≥ 4 mm for Rq = 8–29 μm.
- Effect of shape variation γ (tilt): absolute error increases with γ; large errors for γ > 1° and small Rq.
- Experimental results: Rq = 8.5 → 9.0 μm (+6%); 14.2 → 15.4 μm (+8%); 28.6 → 27.0 μm (−6%); all relative errors < 8.5%.
