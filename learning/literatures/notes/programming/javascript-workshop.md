## Overview
A Packt workshop-style introduction to JavaScript covering language fundamentals, DOM manipulation, asynchronous programming (callbacks, Promises, async/await), Node.js server-side development, testing, and modern frameworks. Exercises use browser developer tools and real web projects throughout.

## Unique contribution
Covers the complete ECMAScript version history through ES9 (2018), explaining which features are available in which environments. Unusually includes chapters on both browser-based and server-side (Node.js) and desktop JavaScript environments, giving readers a panoramic view of where JS runs.

## Methods / results / takeaways
- The history section explains why "Java" appears in "JavaScript" — purely a marketing decision by Netscape/Sun — clarifying common confusion for beginners.
- Arrow functions are introduced as syntactic sugar for function expressions, with explicit notes on how `this` binding differs.
- DOM manipulation exercises build a running to-do list application, providing a tangible project.
- The `fetch` API is preferred over XHR for modern async HTTP, with Promises and `async/await` shown as the clean alternative to callback hell.
- Silent failures in JavaScript (type coercion returning NaN or undefined instead of throwing) are highlighted as a key debugging pitfall.
