## Overview
A Packt workshop-style introduction to Java programming, progressing from Hello World and basic syntax through object-oriented design, collections, exception handling, packages/modules, functional programming with streams and lambdas, concurrency, and network/database access. Exercises and activities are woven throughout every chapter.

## Unique contribution
Unusually broad for a beginner Java book: covers Java 12 enhanced switch syntax alongside classic OOP concepts, and dedicates chapters to the Java Module System (introduced in Java 9) as well as functional programming paradigms. This makes it more current than many intro texts.

## Methods / results / takeaways
- The book emphasizes CLI usage alongside IDE usage from chapter 1, ensuring readers understand the toolchain.
- Local variable type inference (`var`, Java 10+) is introduced as a style convenience, not a replacement for understanding types.
- The collections chapter uses a running AnalyzeInput application to illustrate sorting and iterating through different collection types.
- Exception best-practices chapter distinguishes between checked and unchecked exceptions and explains when not to catch exceptions.
- Java Module System chapter clarifies the `module-info.java` descriptor and `requires`/`exports` directives for encapsulation.
