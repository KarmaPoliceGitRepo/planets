## Overview
A complete set of class notes for Section 8.2 on graphing polar equations, providing many worked examples with step-by-step construction of curves. Curves covered include circles (r = const, r = 2cosθ), spirals (r = θ), cardioids, limaçons, and rose curves of various petal counts.

## Unique contribution
Provides sequential frame-by-frame construction of each polar curve (showing the curve at θ = kπ/6 for increasing k), giving an animated-like understanding of how polar graphs are traced. Includes a matching exercise linking equations to their graphs.

## Key concepts
- Polar coordinates (r, θ)
- Polar curve r = f(θ)
- Circle in polar form
- Archimedean spiral r = θ
- Cardioid r = 1 ± sin θ, r = 1 ± cos θ
- Limaçon r = a + b cos θ
- Rose curves r = sin(nθ), r = cos(nθ)
- Petal count: n petals if n odd, 2n petals if n even

## Methods / results / takeaways
- r = 2 corresponds to a circle of radius 2 centred at origin (r^2 = 4, i.e. x^2 + y^2 = 4).
- r = 2cosθ corresponds to (x-1)^2 + y^2 = 1 (circle of radius 1, centre (1, 0)).
- r = 1 + sinθ traces a cardioid; the cusped point is at r = 0 (θ = 3π/2).
- Rose r = sin(nθ): n odd → n petals (traced once in [0, π]); n even → 2n petals (traced in [0, 2π]).
- Converting between polar and Cartesian: x = r cosθ, y = r sinθ, r^2 = x^2 + y^2.
