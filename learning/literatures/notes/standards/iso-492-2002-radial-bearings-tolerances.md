## Overview
The fourth edition of ISO 492 specifies tolerances for boundary dimensions (bore, outside diameter, ring widths) and running accuracy (radial and axial runout) for radial rolling bearings covered by ISO 15, ISO 355, and ISO 8443. Does not apply to drawn-cup needle roller bearings, airframe bearings, or instrument precision bearings.

## Unique contribution
Uses traditional symbol set (Δdmp, Vdmp, Kea, Sia, etc.) defined in ISO 15241. Tabulates tolerance values across five tolerance classes (Normal, 6, 5, 4, 2 for standard bearings; Normal, 6X, 5, 4, 2 for tapered roller bearings) for bore diameters from sub-1 mm up to 2500 mm. This is the edition superseded by ISO 492:2014.

## Key concepts
- Tolerance classes: Normal, Class 6, 5, 4, 2 (standard); 6X added for tapered rollers
- Single bore diameter deviation Δdmp; variation Vdmp, Vdsp
- Radial runout Kea, Kia; axial runout Sea, Sia; perpendicularity Sd, SD
- Outer ring width deviation ΔCs; inner ring width ΔBs
- Tapered bores (tapers 1:12 and 1:30)
- Assembled bearing width T; effective widths T1, T2 for tapered roller bearings

## Methods / results / takeaways
Tolerance tables (Tables 1–26) cover bore diameters d and outside diameters D with separate tables per tolerance class. For Normal class, inner ring bore d = 18–30 mm: Δdmp = 0/−10 µm, Vdmp = 13 µm, Kia = 13 µm, VBs = 20 µm. Outer ring tolerances are slightly larger at the same diameter. Class 6 tightens Δdmp to 0/−8 µm and Kia = 8 µm. For tapered bores, effective bore deviation is measured at the small end; taper slope deviation ΔSL is specified separately. A key practical point: diameters for diameter series 9, 0, 1 have slightly different Vdsp limits than series 2, 3, 4 in several tolerance classes.
