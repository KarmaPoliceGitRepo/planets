## Overview
Part 1 of the three-part EN 50324 series covering piezoelectric ceramics. This part establishes the terminology, definitions, classification of material groups, vibration mode nomenclature, and reference specimen geometries used throughout the series. It is a prerequisite for understanding Parts 2 and 3.

## Unique contribution
Provides the normative classification of piezoceramics into eight types (100–800) based on composition and application characteristics, with a summary table of key properties (permittivity, Qm, d33, Curie temperature) for each type. The type designations (e.g., Type 100 = hard PZT for high-power ultrasonics, Type 200 = soft PZT for sensors) are the practical vocabulary used across the industry.

## Key concepts
- Ferroelectric ceramic / poling / remanent polarisation
- Curie temperature (ϑC)
- Piezoelectric charge constant dij, voltage constant gij
- Electromechanical coupling factor k (kp, k31, k33, k15, kt)
- Vibration modes: transverse length, radial, thickness extensional, longitudinal length, thickness shear
- Mechanical quality factor Qm
- Ageing / depolarisation
- PZT (lead zirconate titanate) types 100–800
- Barium titanate (Type 400), lead titanate (Type 700), lead metaniobate (Type 800)
- Coordinate axis notation (1, 2, 3 directions; poling along 3)

## Methods / results / takeaways
Defines specimen geometries with dimensional ratios required for each vibration mode (e.g., l/t > 5 for transverse mode, d/t > 10 for radial mode). The type property table shows Type 100 (hard PZT) has Qm ≥ 300 and d33 = 250–400 pC/N; Type 200 (soft PZT) has lower Qm (≥ 100) and higher d33 (400+ pC/N); Type 300 (very hard PZT) has Qm ≥ 800 for maximum power handling. Establishes that all measurements must be taken at least 24 h after poling.
