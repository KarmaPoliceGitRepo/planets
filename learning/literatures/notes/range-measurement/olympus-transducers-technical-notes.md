## Overview
A comprehensive reference guide for ultrasonic transducers covering selection criteria, acoustic properties of materials, near-field calculations, beam characteristics, and transducer types (contact, angle beam, delay line, immersion, dual, shear wave). Includes tables for 30+ materials and pre-computed near-field distances for a range of transducer sizes and frequencies.

## Unique contribution
Single-document practical reference combining material acoustic property tables, near-field distance nomographs, and transducer type selection guidance — including focusing parameters (FPF, PTF, OLF focal length modes) not typically co-located in a single resource.

## Key concepts
- Ultrasonic transducer types (contact, angle beam, delay line, immersion, dual, shear wave)
- Near field distance (N)
- Acoustic impedance
- Sound velocity in materials
- Beam divergence
- Focusing (FPF, PTF, OLF)
- Point target focus (PTF)
- Flat plate focus (FPF)
- Optical limit focus (OLF)
- dB loss / attenuation

## Methods / results / takeaways
- Near field formula: N = D²f / (4c) = D² / (4λ), where D = element diameter, f = frequency, c = sound speed, λ = wavelength.
- Acoustic properties table covers 30+ materials (metals, plastics, composites, biological tissue): longitudinal velocity, shear velocity, acoustic impedance.
- Pre-computed near-field distances for transducers: 0.5–25 MHz, 0.125"–1.5" diameter.
- Maximum focal length for PTF mode: 0.8 × N.
- Transducer type selection guidance: contact for direct coupling, immersion for scanning, delay line for hot/rough surfaces, dual for thin sections (eliminates dead zone), angle beam for weld inspection.
- Practical reference for transducer selection and setup in NDT/NDE applications.
