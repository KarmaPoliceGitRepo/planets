# Temperature Stability of Guided Wave SHM: Optimal Baseline Selection

## Overview
Investigates temperature-induced variability in guided-wave structural health monitoring (SHM) signals and proposes the Optimal Baseline Selection (OBS) technique, which selects from a pre-acquired baseline library the signal recorded at the temperature closest to the current monitoring temperature, improving damage detection sensitivity.

## Unique contribution
First systematic demonstration that OBS from a baseline library can reduce temperature-induced false alarms by ~20 dB in guided-wave SHM, compared to using a single fixed baseline.

## Key concepts
- Guided wave SHM
- Optimal Baseline Selection (OBS)
- Temperature compensation
- Baseline library
- Damage index
- Lamb waves

## Methods / results / takeaways
- Experiments on aluminium plate, temperature range 22–32°C.
- OBS reduces residual signal amplitude by ~20 dB compared to single baseline, enabling detection of smaller defects.
- Damage index (DI) based on residual energy after baseline subtraction.
- Temperature sensitivity: even 1–2°C variation significantly degrades single-baseline detection; library selection mitigates this.
- Sets foundation for BSS (Baseline Signal Stretch) and combined OBS+BSS methods in subsequent works.
