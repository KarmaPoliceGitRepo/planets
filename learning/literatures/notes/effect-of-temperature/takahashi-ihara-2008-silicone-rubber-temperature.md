# Ultrasonic Velocity in Silicone Rubber vs Temperature

- **Source:** 2008 - Takahashi Ihara - silicone rubber temperature.pdf

## Overview
Measures longitudinal ultrasonic velocity in silicone rubber as a function of temperature and demonstrates that this V(T) relationship can be used to measure temperature inside a silicone rubber component via pulse-echo, using the bisection algorithm.

## Unique contribution
Provides V(T) calibration for silicone rubber and demonstrates pulse-echo ultrasonic thermometry in a polymer material, extending the bisection-algorithm technique beyond metals to compliant soft materials.

## Key concepts
- Silicone rubber
- Longitudinal velocity vs temperature
- Polymer ultrasonic thermometry
- Bisection algorithm
- Pulse-echo

## Methods / results / takeaways
- V(T) calibration for silicone rubber: V = −2.71·T + 1072.5 m/s (strong negative slope, ~2.7 m/s/°C).
- Silicone rubber shows much larger fractional velocity change per °C than metals.
- Bisection algorithm applied with this V(T) to reconstruct temperature profile in a silicone rubber block.
- Agreement with thermocouple within ±5°C over 20–80°C range.
- Large dV/dT makes silicone rubber a highly sensitive thermometric material but requires precise calibration.
