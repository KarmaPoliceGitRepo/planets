# Ultrasonic Temperature Measurement in Polymer Extrusion Using Clad Buffer Rod

## Overview
Applies a clad buffer-rod ultrasonic probe (steel core, stainless steel cladding) to measure melt temperature and velocity in a polymer extrusion process. Incorporates a Kalman filter to improve real-time temperature estimates from pulse-echo TOF data.

## Unique contribution
First application of a clad buffer rod combined with Kalman filtering for real-time ultrasonic thermometry in polymer extrusion, demonstrating improved noise immunity and dynamic response.

- Clad buffer rod

## Methods / results / takeaways
- Clad buffer rod: steel core with stainless steel outer sleeve eliminates mode conversion at rod/melt interface.
- V(T) calibration for steel buffer rod (longitudinal waves): V_L = −0.6571·T + 5951 m/s.
- Kalman filter applied to TOF sequence to smooth noise and track temperature ramp.
- Measured melt temperatures agree with thermocouple within ±2°C at steady state.
- Buffer rod approach keeps transducer at room temperature while probe tip contacts melt at ~200–300°C.
