## Overview
Proposes a new "systems model" for thickness-mode piezoelectric transducers that represents the transducer as a linear time-invariant system described by its transfer function, combining both the electrical and acoustic domains in a unified signal-processing framework. Uses z-transform and digital signal processing concepts to derive the impulse response and frequency response of transducers with backing and matching layers.

## Unique contribution
Reframes transducer modelling as a linear systems / signal processing problem, enabling systematic application of DSP techniques (z-transform, digital filter design) to transducer design. This approach complements the traditional equivalent circuit (Mason, KLM) methods with a more computationally direct transfer function formulation.

## Key concepts
- Systems model of piezoelectric transducer
- Transfer function approach (vs equivalent circuit models)
- z-transform representation
- Thickness mode transducer
- Impulse response and pulse-echo response
- Backing and front loading effects on transfer function
- Digital signal processing applied to transducer design
