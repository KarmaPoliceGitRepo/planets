# Classification and Characterization of PVDF Sensors Using Finite Element Simulation

## Overview
Uses finite element (FE) simulation to classify PVDF piezoelectric sensors by their mechanical and piezoelectric behaviour. Focuses on d31 mode characterisation, extracting Young's modulus and d31 coefficient from simulated and measured impedance/strain data.

## Unique contribution
Provides a systematic FE-based methodology for PVDF sensor classification that links mechanical parameters to electrical response, useful for calibrating and quality-sorting PVDF film transducers prior to deployment.

## Key concepts
- PVDF finite element simulation
- d31 piezoelectric coefficient
- Young's modulus PVDF
- Sensor classification
- Impedance characterisation
- P(VDF-TrFE) copolymer
- Poling direction
- Mechanical vs. piezoelectric coupling

## Methods / results / takeaways
- FE model built in ANSYS/COMSOL using PVDF material properties; d31 mode (in-plane strain, through-thickness voltage)
- Young's modulus of PVDF: 3–4 GPa (frequency-dependent; higher for TrFE copolymer)
- d31 coefficient: 20–30 pC/N for standard PVDF films; higher for TrFE copolymer grades
- Classification approach: compare measured impedance spectrum with FE predictions; assign to material class
- Demonstrates sensitivity of classification to poling history and film thickness variations
- Results validated against manufacturer datasheets (Measurement Specialties / TE Connectivity PVDF films)
- Applicable to quality control in PVDF transducer manufacturing and array element screening
