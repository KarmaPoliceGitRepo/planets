# PVDF Multilayer Transducers for Ultrasonic Applications

## Overview
Presents design and fabrication of multilayer PVDF transducers (switchable Barker-coded transducers, SBCT) for broadband ultrasonic applications. Uses multiple PVDF layers with coded polarization to encode and decode ultrasonic pulses, improving SNR without sacrificing bandwidth.

## Unique contribution
Introduces the Switchable Barker-Coded Transducer (SBCT) concept for PVDF multilayer stacks, where individual layers can be electrically switched to change the coding pattern. Achieves SNR improvement equivalent to a Barker code gain while retaining pulse-echo operation capability.

## Methods / results / takeaways
- Barker codes are binary phase-coding sequences (e.g., +1 +1 +1 -1 -1 +1 -1 for 7-bit) that have low autocorrelation sidelobes
- SBCT: stack of PVDF layers, each independently polable/switchable; layer polarities set to match Barker code pattern
- Transmitted pulse is naturally coded; received echo decoded by matched filter, compressing pulse and improving SNR by ~N (number of layers)
- Demonstrated at 2–5 MHz with 4–8 PVDF layers; SNR improvement ~6–9 dB for 7-bit code
- Bandwidth preserved: PVDF's low Q means individual layers are broadband; stacking adds coding without narrowing bandwidth
- Fabrication: individual PVDF films (9–28 µm) laminated with conductive adhesive; poling controlled per layer
- Switching achieved via external relay network to reconfigure layer connections
- Applicable to NDE where SNR is critical (e.g., highly attenuating materials, air coupling)
