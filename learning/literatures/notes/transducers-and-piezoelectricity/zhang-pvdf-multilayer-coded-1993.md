## Overview
Conference paper (IEEE Ultrasonics Symposium 1993) introducing Barker-coded multilayer PVDF transducers. Demonstrates that polarization-coded PVDF stacks improve SNR via pulse compression while maintaining the broadband response inherent to PVDF films.

## Unique contribution
Early experimental validation of the Switchable Barker-Coded Transducer (SBCT) concept in PVDF. Shows that a 7-element Barker code applied to a multilayer PVDF stack yields ~2× bandwidth improvement over comparable single-crystal PZT transducers with matched SNR.

## Methods / results / takeaways
- 7-element Barker code (+1+1+1-1-1+1-1) applied to polarity of individual PVDF layers in a stack
- Transmitted waveform naturally Barker-coded; received echo decoded by matched filter → pulse compression
- 7-layer PVDF stack demonstrated at ~5 MHz; bandwidth approximately twice that of single PZT element
- SNR gain ~8.5 dB from 7-element code compression
- SBCT operates in pulse-echo mode by electrically switching layer connections before and after transmit
- Confirms PVDF's low acoustic impedance (~4 MRayl) allows stacking without bandwidth degradation
- Forerunner to Zhang/Lewin 1997 paper with full analytical treatment
