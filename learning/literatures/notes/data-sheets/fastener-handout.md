## Overview
A comprehensive course handout on mechanical fasteners for engineering drawing representation. Covers screw thread geometry, Unified and Metric thread standards, local notes (callouts) for holes and threaded features, cap/machine/set screw specifications, head types, and appendices with fastener head dimension tables, recommended torque tables, bolt grade markings, and drill size decimal equivalents.

## Unique contribution
Provides a practical, all-in-one reference for specifying threaded fasteners on engineering drawings, including the UNC/UNF tap chart with tap drill diameters, metric thread compatibility between M and MJ series, and the ASME Y14 symbology for counterbores, countersinks, and spotfaces. Includes bolt grade markings and strength charts for Grades 2, 5, 8, and ISO 8.8/10.9.

## Key concepts
- Unified thread series (UNC, UNF, UNEF) and metric (M, MJ) threads
- Thread angle, pitch, lead, major/minor/pitch diameters
- Class fit (1A/B, 2A/B, 3A/B; ISO 6H/6g)
- Cap screws, machine screws, set screws
- Head types (hex, fillister, flat, oval, round, socket)
- Counterbore, countersink, spotface callouts
- Tap drill diameter and thread engagement length
- Pre-load and seating torque
- Set screw holding power and point types
- Left-hand threads

## Methods / results / takeaways
- Thread angle effect: lower thread angle → higher load capacity; square and buttress threads used for power transfer.
- UNC is most common general-purpose thread; UNF offers higher shear load capacity and better torque-locking for same diameter.
- UNEF is for aerospace and applications with minimum thread engagement.
- Minimum thread engagement: 1–1.5× major diameter for brass/cast iron/aluminum; 0.75–1× for steel.
- Set screw holding power increases ~30% with a second screw diametrically opposed; ~75% with a second screw inline.
- ISO metric class 6H/6g is equivalent to ASME class 2B/2A; 4H5H/4h6h ≈ class 3B/3A.
- Recommended torques table covers Grades 5, 8, stainless, bronze, brass from #4 through 1 inch.
