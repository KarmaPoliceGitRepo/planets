# Absolute Grain Size Determination from Laser Ultrasonic Measurements Without Calibration

## Overview
A paper demonstrating calibration-free absolute grain size measurement using LUS and the Stanke-Kino unified theory. Applied to five commercial low-carbon steels at frequencies 12.5–37.5 MHz, extracting grain sizes without any reference specimen by using single-crystal elastic constant data for the material system.

## Unique contribution
Demonstrates that the Stanke-Kino theory, combined with known single-crystal elastic constants, allows absolute grain size extraction from attenuation measurements without any empirical calibration on reference specimens of known grain size — a key advance for practical industrial deployment.

## Key concepts
- Absolute grain size measurement

## Methods / results / takeaways
- Five low-carbon steel grades annealed to different grain sizes (10–60 µm verified by metallography).
- LUS attenuation measured at 12.5, 25, 37.5 MHz from pulse amplitude ratios of successive echoes.
- Grain size inverted from α(f) by fitting Stanke-Kino model with single-crystal constants for α-Fe.
- Errors: ~10% relative error across all five grades (systematic bias ~5% attributed to texture effects).
- Grain size range 10–60 µm well-covered; below 10 µm, scattering at 37.5 MHz approaches noise floor.
- Fully automated algorithm: no operator intervention; suitable for on-line integration.
- Comparison with metallographic ASTM grain size: consistent within stated error bars.
- Key finding: elastic constant uncertainty (±2%) contributes ~6% uncertainty to grain size result.
