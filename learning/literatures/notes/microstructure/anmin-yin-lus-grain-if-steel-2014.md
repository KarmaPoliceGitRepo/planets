# Laser Ultrasonics for Grain Size Measurement in IF Steel

## Overview
An application paper using laser ultrasonics (LUS) to measure grain size in interstitial-free (IF) steel samples. Employs the Rayleigh scattering regime formula α = Cr·D³·f⁴, calibrating the material constant Cr from one specimen and applying it to others. Investigates accuracy for grain sizes ranging from 5 to 80 µm.

## Unique contribution
Demonstrates a practical single-calibration LUS grain-size measurement system for IF steel, showing achievable accuracy and identifying the breakdown for large grains; proposes an on-line measurement system for hot rolling mills based on the approach.

## Methods / results / takeaways
- LUS system: pulsed Nd:YAG generation laser; interferometric detection; non-contact.
- Four IF steel grades (IF1–IF4) with grain sizes 5–80 µm prepared by different annealing conditions.
- Cr calibrated from IF1 (5.2 µm reference); applied to IF2, IF3, IF4.
- Results: IF2 (12 µm) good agreement; IF3 (35 µm) 15% error; IF4 (80 µm) 34% error.
- Error increases for large grains because: (a) Rayleigh regime assumed but grain size approaches stochastic regime; (b) multiple scattering effects.
- Proposed correction: use Stanke-Kino full model instead of simple Rayleigh formula for large grains.
- On-line system concept: LUS applied to hot strip mill at 900°C; non-contact enables austenite grain monitoring during processing.
