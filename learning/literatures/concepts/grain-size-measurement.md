# Grain Size Measurement
**Aliases:** grain size; mean grain diameter; ASTM grain size number; prior austenite grain size; PAGS; ultrasonic grain sizing; grain size distribution; grain size by ultrasonics; metallographic grain size; lineal intercept method; McQuaid-Ehn test

**Definition:** Grain size characterises the scale of the crystalline microstructure in polycrystalline metals, typically expressed as mean equivalent diameter D or ASTM grain size number G. Fine grain size improves toughness and fatigue resistance (Hall-Petch strengthening); coarse grains increase creep resistance at high temperatures. Metallographic methods (lineal intercept, ASTM E112) require destructive sectioning. Ultrasonically, grain size is non-destructively estimated from the frequency dependence of attenuation: in the Rayleigh regime α ∝ f⁴D³, allowing D to be extracted from measured α(f) at multiple frequencies.
