# Hughes–Kelly Model
**Aliases:** Hughes-Kelly equations; Hughes and Kelly 1953; classical acoustoelastic equations; Man-Lu model; Iwashimizu-Kubomura theory; five-constant elasticity theory

**Definition:** The Hughes–Kelly model (1953) provides the first systematic acoustoelastic equations relating seven principal ultrasonic wave velocities in an isotropic prestressed solid to the applied stress, using Murnaghan's three TOE constants (l, m, n). (Two equations cover hydrostatic pressure; five cover uniaxial tension, per Hughes & Kelly 1953.) The velocity–stress relations take the form ρ₀v² = c_ij + (linear combination of Lamé and TOE constants) × stress components, where c_ij are the unstressed second-order elastic constants. Extensions include the Man–Lu model (1987) for textured polycrystals, and the Iwashimizu–Kubomura theory which accounts for initial finite strain. Per the conflict rule: Hughes–Kelly (journal, 1953) is preferred over textbook derivations for the primary equations.
