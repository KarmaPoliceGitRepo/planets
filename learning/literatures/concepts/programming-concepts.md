# Programming Concepts
**Aliases:** object-oriented programming; OOP; inheritance; mixin; duck typing; lambda expression; arrow function; async/await; Promise; Stream API; Java Module System; Node.js; ECMAScript; generics; pointer; struct; dynamic memory allocation; stack vs. heap; preprocessor directive; metaprogramming; NumPy; scikit-learn; pandas; Matplotlib; Bokeh; gcc; confusion matrix; Viterbi decoder; state machine architecture; producer-consumer; reentrant VI; TDMS; VISA

**Definition:** Object-oriented programming (OOP) organises code around objects encapsulating state and behaviour; inheritance shares behaviour through class hierarchies; mixins add behaviour without inheritance. Duck typing (Python, Ruby) tests interface compatibility at runtime. Lambda/arrow functions are anonymous inline functions; async/await is syntactic sugar over Promises for non-blocking I/O. In Python, NumPy provides vectorised array computation; scikit-learn provides ML algorithms; pandas provides dataframe-based data wrangling; Matplotlib and Bokeh handle visualisation. A confusion matrix tabulates true/false positive/negative predictions. The Viterbi decoder finds the maximum-likelihood path through a trellis (HMM decoding).
