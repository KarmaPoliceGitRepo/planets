# Plateau Honing Parameters
**Aliases:** Rk; Rpk; Rvk; Mr1; Mr2; Rmr; Rpq; Rvq; Rmq; reduced peak height; core roughness depth; reduced valley depth; material ratio; bearing ratio curve; Abbott-Firestone curve; ISO 13565; Rk parameters

**Definition:** ISO 13565-2 defines five parameters for plateau-honed surfaces, derived from the bearing ratio (Abbott-Firestone) curve: Rk (core roughness depth — the working contact zone), Rpk (reduced peak height — material to be removed by running-in), Rvk (reduced valley depth — oil-retention capacity; rule of thumb ≥ 0.6 µm), Mr1 (material ratio at plateau-valley transition), and Mr2 (material ratio at valley floor). ISO 13565-3 adds probability-model parameters Rpq, Rvq, Rmq. These replace Ra for functional specification of engine bores: low Rpk minimises break-in wear, adequate Rvk ensures lubrication retention, and Rk controls load-bearing behaviour.
