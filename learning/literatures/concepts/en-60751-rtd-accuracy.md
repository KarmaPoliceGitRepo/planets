# EN 60751 RTD Accuracy Classes

**Aliases:** IEC 60751; DIN EN 60751; RTD tolerance classes; Class AA accuracy; Class A accuracy; Class B accuracy; tolerance class AA; tolerance class A; tolerance class B

**Definition:** EN 60751 (identical to IEC 60751) is the international standard defining the resistance-temperature relationship and accuracy classes for platinum RTDs. The tolerance classes are: Class AA (highest): ±(0.10 + 0.0017|t|) °C; Class A: ±(0.15 + 0.002|t|) °C; Class B: ±(0.30 + 0.005|t|) °C. Class B is the standard for most industrial Pt100 applications; Class A is used for tighter requirements; Class AA for calibration and laboratory use. The standard also defines the Callendar–Van Dusen constants and wire configurations. The tolerances apply at the sensor element only (not including lead resistance or transmitter error).
